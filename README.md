## What is this?

This is the public marketing site for [FatedFortress](https://github.com/NeuralDraftLLC/fatedfortress) — an AI-scoped task marketplace with Stripe escrow, zero ambiguity, and guaranteed payouts.

## The Product

FatedFortress turns messy project ideas into machine-enforced contracts:

1. **Scope** — GPT-4o decomposes a natural-language brief into JSON task specs with exact rules for "done"
2. **Fund** — Stripe payment intents lock funds on task claim (24-hour soft lock, manual capture)
3. **Verify** — Cloudflare Worker validates file MIME, size, and content before entering review queue
4. **Release** — Host approves or 48h auto-release; every decision immutably logged

## Technology Stack

- **Frontend:** Vite SPA, vanilla TypeScript, Swiss-brutalist dark UI
- **Auth:** Supabase (magic link + Google OAuth, role-gated)
- **Database:** PostgreSQL via Supabase (RLS, pg_cron)
- **AI:** GPT-4o edge function for task scoping
- **Payments:** Stripe Connect (escrow via payment intents with manual capture)
- **Realtime:** Y.js + y-webrtc for live collaborative review
- **Verification:** Cloudflare Worker (asset MIME/size/content validation)
- **Storage:** Supabase Storage (presigned URLs, private buckets)
